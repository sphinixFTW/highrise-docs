<Step>
## Installing Unity Hub
<Callout title="Step One">
Unity Hub is the key to utilizing Highrise Studio Hub.
</Callout>

<Tabs items={['Installing Unity Hub', 'What is Unity Hub?']} defaultIndex={0}>
<Tab value="Installing Unity Hub">
 To get started in creating your very first world, you first have to install the Hub by clicking the button below.
<Cards>
  <Card title="Install Unity Hub" href="https://unity.com/download" className="card" />
</Cards>
</Tab>

<Tab value="What is Unity Hub?">
- Unity Hub is a tool that helps you manage your Unity projects and installations.
- It's essential for using Highrise Studio Hub because it allows you to access and organize your Highrise projects efficiently.
- With Unity Hub, you can easily install different versions of Unity and switch between them as needed.
- It provides a convenient way to discover and access the Highrise Studio Hub, where you can create your own worlds and experiences.
</Tab>
</Tabs>
</Step>

<Step>
## Installing Studio Hub
<Callout title="Step Two">
Highrise Studio Hub is the key to create 3D worlds in Highrise
</Callout>

<Tabs items={['Installing Studio Hub', 'What is Studio Hub?']} defaultIndex={0}>

<Tab value="Installing Studio Hub">
Step two is to install the official Studio Hub, which serves as a central platform for creating games on the Highrise platform.
<Cards>
  <Card title="Install Studio Hub" href="https://create.highrise.game/highrise-studio" className="card" />
</Cards>
</Tab>

<Tab value="What is Studio Hub?">
- Highrise Studio Hub is a central platform for creating games and experiences on the Highrise platform.
- It serves as the main workspace where you can access tools, resources, and manage your projects. 
- Highrise Studio Hub simplifies the process of developing and publishing games for various devices, offering a user-friendly interface and a comprehensive set of features. 
- Whether you're a beginner or an experienced developer, Highrise Studio Hub provides everything you need to bring your ideas to life and share them with others worldwide.
</Tab>
</Tabs>
</Step>
